Exercício 1
Programa 1  -	2024_08_19_Exercicio1_1.js
Função 	    -	Cria Array, Verifica Termos Repetidos e Cria um novo array sem os números repetidos
Descrição   -	Através das informações fornecidas pelo usuário um array é criado, é verificado qual o termo mais se repete e cria um novo array sem os termos repetidos
Exemplo     -	Um array de tamanho 5 com termos de valor máximo 10 e valor mínimo 0 e tem como resposta esperada [3,8,3,9,10], o valor que mais se repete 3, array sem números repetidos [3,8,9,10]
